CprE 288 - Intelligent Mapping Roomba

Throughout the embedded systems class, I had developed a modular library that reliably controlled the Roomba. Using this modularity, basic telemetry data could be collected and transmitted (initially to a console and then to a GUI) and movement algorithms to reach a target while avoiding obstacles. With this competed ahead of time, the project's goal could be ambitious: Control a Roomba with a DualShock4 controller using nothing more than a GUI sent by the telemetry on the bot while also making use of autonomous algorithms. The Roomba in real time would send data to update a live map for the pilot using the controller. The controller moved the Roomba in the way a video game would move a character. The pilot could use the right joystick of the controller as well to move a target on the map for the Roomba to autonomously drive to. This project was an absolute joy.
